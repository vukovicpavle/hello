# Hello

Simple say hello to the world.

## Installation

```bash
npm install @vukovicpavle/hello
```

## Usage

```javascript
const hello = require("@vukovicpavle/hello");

hello("World");
```

## API

### hello(name)

Say hello to the name provided.

#### Parameters

| Name | Type  | Description           |
| ---- | ----- | --------------------- |
| name | `any` | Name to say hello to. |
